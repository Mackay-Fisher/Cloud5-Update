Perfect Snake

Perfect Snake was my first introduction to machine learning and focused on the use of DNQ agents as well as randomly generated actions to create a desired path to capture randomly generated elements on a created playing board. Then created a systematic reward system in order to showcase a properly managed DNQ reward system, I utilized the Pytorch library in order to unitize an agent class on a pygame snake game that I created my self in order to give access to the possible controls. The used Q-learning agent in order to create a systems to train both short term and long-term training adjusting shallow neural networks with as well as depth for further optimization. This created a reinforcement learning model that tracked both short term and long term decision making to create optimal decision trees. Then utilized Matplotlib to showcase real timed training with scores and iteration to show progression to higher and higher scores as the model progressed.

AI Mario

In the next step towards machine learning I began to work with a reinforcement learning model, this was particularly based on the open ai gym model for ai mario, I followed tutorial videos on how to use stable baselines before then working on creating my own algorithm. I started by using vector stacking to view the imaging of the level and well as a finite move list to help build an optimal path decision tree. I then worked with a consistent reward system based on distance to help motivate the model to progress throughout the level. Then as the iteration continued I stored them in long term and short term memory to build a width and deep neural network to optimize efficiency. I then utilized the Stable Baseline PPo model to execute the callback of these iterations and add them to my model which then led to a trial process that was able to finish the first level of mario.
